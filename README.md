# Portfolio - Guillaume Bertil
 
Site vitrine / CV personnel développé avec React, Vite et DaisyUI.
 
## 🚀 Démo
 
[guillaumebertil.vercel.app](https://guillaumebertil.vercel.app)
 
## 📋 Sections
 
- **Hero** - Présentation et accroche
- **À propos** - Parcours, motivations et objectif
- **Compétences** - Stack technique avec niveaux
- **Portfolio** - Projets réalisés
- **Contact** - Formulaire de contact et liens
 
## 🛠️ Stack technique
 
- [React](https://react.dev/) - Framework JavaScript
- [Vite](https://vitejs.dev/) - Bundler
- [DaisyUI](https://daisyui.com/) - Composants UI (thème Night)
- [Tailwind CSS](https://tailwindcss.com/) - Utilitaires CSS
- [Devicon](https://devicon.dev/) - Icônes technologies
- [Formspree](https://formspree.io/) - Gestion du formulaire de contact
 
## 📦 Installation
 
```bash
# Cloner le projet
git clone https://github.com/guillaumebertil/portfolio.git
 
# Accéder au dossier
cd portfolio
 
# Installer les dépendances
npm install
 
# Lancer le serveur de développement
npm run dev
```
